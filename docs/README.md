# IEUMBOT 문서 인덱스

이 디렉터리는 IEUMBOT 프로젝트의 기획, 아키텍처, API, 보안, 테스트, 운영 기준을 번호순 마크다운 문서로 정리하기 위한 공간입니다. 각 문서는 구현 전에 합의해야 할 기준과 결정 사항을 명확히 기록하는 것을 목표로 합니다.
